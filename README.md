# 教务管理系统 python256

## 项目概述

教务管理系统是一个基于 Flask 框架的 Python 应用，旨在实现高校教务管理的基本功能，包括老师、学生的角色区分，以及课程开设、成绩管理、资源文件上传等。

## 技术栈

- 后端：Python 3.x
- Web 框架：Flask
- 数据库：SQLite

## 功能模块

### 用户管理

- 登录注册

### 教师模块

- 开设课程
- 录入成绩
- 上传教学资源文件
- 课程管理

### 学生模块

- 选课
- 查看个人信息
- 查看课程信息

## 系统角色

- 教师
- 学生

## 运行环境

- Python 3.x
- Flask
- SQLite

## 部署步骤

1. 安装 Python 3.x 和 Flask
2. 克隆或下载项目代码
3. 安装项目依赖
4. 初始化数据库
5. 运行应用
